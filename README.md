# Python Recursion — Basic to Advanced

This repository contains a single example file demonstrating recursion techniques in Python, from basic patterns to advanced topics and common interview problems.

## Contents
- [PYTHON RECURSION - BASIC TO ADVANCED.py](PYTHON%20RECURSION%20-%20BASIC%20TO%20ADVANCED.py): a single script with examples and explanations (factorial, Fibonacci, recursion patterns, backtracking, memoization, tail recursion, practice problems).

## Prerequisites
- Python 3.8 or newer

## Usage
Run the script directly with Python:

```
python "PYTHON RECURSION - BASIC TO ADVANCED.py"
```

Read through the file to see examples and notes. Each section is self-contained and can be adapted into modules or tests.
